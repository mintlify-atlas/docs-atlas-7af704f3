---
title: "ready"
description: "List issues that are ready to work on (no blocking dependencies)"
---

## Syntax

```bash
tracer ready [OPTIONS]
```

## Options

<ParamField path="--priority" type="integer">
  Filter by priority level (0-4, where 0 is highest priority)
</ParamField>

<ParamField path="--assignee" type="string">
  Filter by assignee email or name
</ParamField>

<ParamField path="--limit" type="integer">
  Maximum number of results to return
</ParamField>

<ParamField path="--json" type="boolean">
  Output result in JSON format
</ParamField>

## Examples

<CodeGroup>

```bash Basic Usage
# Show all ready work
tracer ready
```

```bash Filter by Priority
# Show only highest priority ready work
tracer ready --priority 0
```

```bash Filter by Assignee
# Show ready work assigned to Alice
tracer ready --assignee alice@example.com
```

```bash Limit Results
# Show top 5 ready issues
tracer ready --limit 5
```

```bash Combined Filters
# Show top 3 highest priority issues for Bob
tracer ready --priority 0 --assignee bob@example.com --limit 3
```

```bash JSON Output
# Get structured JSON for programmatic use
tracer ready --json
```

</CodeGroup>

```
  Created: 2025-03-04 11:00 | Updated: 2025-03-04 11:00

  Status: open
  Assignee: alice@example.com
  Created: 2025-03-04 12:00 | Updated: 2025-03-04 12:00
```

```
No ready work found
```
